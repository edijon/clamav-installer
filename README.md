# clamav on-access scanning installer

![ci_workflow](https://github.com/edijon/clamav-installer/actions/workflows/blank.yml/badge.svg)

- `ClamAV` website : https://www.clamav.net/
- Developped with bash for Ubuntu 20.04 LTS
- You are free to use it on your own risk

## Usage

### Ubuntu 20.04 LTS

- Get this repository
    ```bash
    git clone --depth=1 https://github.com/edijon/clamav-installer.git
    ```
- Run `install.sh`
    ```bash
    cd clamav-installer
    ./install.sh
    ```
