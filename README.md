# clamav on-access scanning installer

Developped with bash for Ubuntu 20.04 LTS

## Usage

### Ubuntu 20.04

- Get this repository
    ```bash
    git clone --depth=1 ssh://git@github.com/edijon/clamav-tools.git
    ```
- Run `install.sh`
    ```bash
    cd clamav-installer
    ./install.sh
    ```
